# Meal Planner

**[Live demo →](https://meal-planner-taupe-gamma.vercel.app)**

A 4-week dinner plan generator built with React + Vite. Pick your cuisines, set how many nights you cook, and get a full plan with a shopping list in one click.

## Features

- **5 cuisines** — Italian, Asian, Mexican, Mediterranean, American (35 recipes total)
- **Leftover nights** — automatically references batch-friendly meals from the same week
- **Proteins on hand** — suggest meals that use up what's in the fridge
- **Per-meal controls** — thumbs up/down, swap, eating-out toggle, reorder arrows
- **Custom recipes** — add your own to any week
- **Shopping list** — auto-generated by category (Protein, Produce, Pantry, Other), excludes leftovers and eating-out nights

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Building for production

```bash
npm run build
```

Output goes to `dist/`.

## Deploying to Vercel

1. Push to a GitHub repository.
2. Import the repo in [vercel.com/new](https://vercel.com/new).
3. Framework preset: **Vite** — Vercel auto-detects it.
4. Click **Deploy**. Done.

The `vercel.json` at the project root handles client-side routing rewrites.

## Project structure

```

  data/

  utils/
```
